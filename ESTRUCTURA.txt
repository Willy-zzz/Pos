FUNCIONALIDADES PRINCIPALES:
============================

1. AUTENTICACIÓN
   - Login con usuario y contraseña
   - Control de sesiones
   - Roles: Administrador y Usuario

2. GESTIÓN DE CLIENTES
   - Crear, editar, eliminar clientes
   - Búsqueda y filtrado
   - Datos: nombre, teléfono, email, dirección, RFC

3. GESTIÓN DE PRODUCTOS/SERVICIOS
   - CRUD completo de productos
   - Categorías
   - Precios y tipos (producto/servicio)

4. SISTEMA DE VENTAS (POS)
   - Interfaz tipo punto de venta
   - Carrito de compras
   - Selección de cliente
   - Dos métodos de pago: Efectivo y Tarjeta
   - Cálculo automático de cambio
   - Creación rápida de clientes

5. TICKETS Y CONTRATOS
   - Generación automática de tickets
   - Generación de contratos personalizados
   - Impresión térmica (80mm)
   - Impresión en formato A4
   - Variables dinámicas en plantillas

6. REPORTES
   - Estadísticas de ventas
   - Filtros por fecha
   - Exportación a Excel
   - Gráficas de ventas

7. CONFIGURACIÓN
   - Datos del negocio
   - Gestión de plantillas de contratos
   - Gestión de usuarios
   - Configuración de impresión


TECNOLOGÍAS UTILIZADAS:
=======================

Backend:
- PHP 7.4+
- CodeIgniter 3.1.13
- MySQL 5.7+

Frontend:
- Bootstrap 5.3
- jQuery 3.6
- Font Awesome 6.4
- Chart.js (para gráficas)

Librerías:
- TCPDF (generación de PDFs)
- PHPExcel (exportación a Excel)


REQUISITOS DEL SERVIDOR:
========================

- PHP >= 7.4
- MySQL >= 5.7
- Apache con mod_rewrite
- Extensiones PHP:
  * mysqli
  * gd
  * mbstring
  * json
  * curl
